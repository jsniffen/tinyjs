TinyJS

TinyJS is a simple Javascript framework.

Elements

You can create elements using the element function.

import { element } from "./tiny.js" const button = element("button", { textContent: "Click me!" })

Web Components

You can register web components using the register function.

import { element, register } from "./tiny.js" register("tiny-app", root => { return element("div", { textContent: "hello world" }) })

State

You can create stateful components using the state function.

import { element, state } from "./tiny.js"; const counter = () => { const [onCount, setCount] = state(0) const button = element("button", { onclick: setCount(count => count + 1) }) const div = element("div") onCount(count => { div.textContent = `Count: ${count}` }, div) return element("div", {}, div, button ) }
